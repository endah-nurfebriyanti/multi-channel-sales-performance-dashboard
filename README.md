An interactive **Business Intelligence Dashboard** built using **Google Looker Studio** to analyze sales performance across multiple e-commerce marketplaces. The dashboard integrates transactional data from **Shopee, Tokopedia, Lazada, and Blibli**, enabling users to monitor key performance indicators (KPIs), compare marketplace performance, evaluate refund trends, and identify high-performing products to support data-driven decision making.

# 📖 Business Problem

Businesses that sell products through multiple online marketplaces often generate thousands of transactions every day. However, without an integrated dashboard, it becomes difficult to monitor sales performance, compare marketplace contributions, identify top-selling products, and evaluate refund activities.

Decision-makers need a centralized dashboard that provides real-time insights into sales performance and supports faster, data-driven business decisions.

This project addresses those challenges by transforming raw transactional data into an interactive Business Intelligence dashboard.

---

# 🎯 Project Objectives

The dashboard was developed to:

- Monitor overall sales performance.
- Compare revenue across multiple marketplaces.
- Track daily sales trends.
- Analyze refund values and refund rates.
- Identify top-performing products.
- Evaluate category performance.
- Monitor Average Order Value (AOV).
- Support business decisions using interactive dashboards.

This project uses a **synthetic multi-channel sales dataset** containing transactions from four online marketplaces.

| order_id | Unique transaction ID |
| tanggal_order | Order date |

| produk_id | Product ID |
| produk_nama | Product name |
| kategori | Product category |
| qty | Quantity sold |
| harga_satuan | Unit price |
| total_order | Revenue |
| diskon | Discount |
| ongkir | Shipping cost |
| status | Order status |
| refund_status | Refund status |
| refund_amount | Refund value |

# 💡 Storytelling & Key Findings

The dashboard provides several important business insights.

### 🛒 Marketplace Performance

Marketplace comparison enables decision-makers to identify which sales channel contributes the highest revenue and which marketplaces require additional marketing strategies.

---

### 📈 Sales Trend

Monitoring daily revenue trends helps businesses identify sales fluctuations, promotional impacts, and seasonal purchasing behavior.

---

### 📦 Product Performance

Product-level analysis highlights the products generating the highest revenue and identifies categories that contribute most to total sales.

---

### 🔄 Refund Monitoring

Refund analysis enables businesses to monitor refund values and detect operational issues that may negatively affect customer satisfaction.

---

### 💰 Customer Spending

Average Order Value (AOV) provides insights into customer purchasing behavior and helps evaluate pricing strategies.

# 🚀 Business Recommendations

Based on the dashboard analysis, several strategic recommendations can be proposed:

- Focus marketing campaigns on the highest-performing marketplaces.
- Improve promotional strategies for lower-performing sales channels.
- Prioritize inventory planning for top-selling products.
- Monitor refund trends regularly to reduce operational losses.
- Optimize pricing and discount strategies to increase Average Order Value.
- Continue monitoring KPIs using interactive dashboards to support data-driven decision making.
